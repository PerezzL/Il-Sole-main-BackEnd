# Il-Sole - Sistema de Producción

Sistema de gestión de producción de alimentos con formularios y gestión de registros.

## Estructura

- `BackEnd/` - API REST con Express y PostgreSQL
- `FrontEnd/` - Aplicación React con Vite

## Instalación

### Backend
```bash
cd BackEnd
npm install
npm run setup-all
npm start
```

### Frontend
```bash
cd FrontEnd
npm install
npm run dev
```

## Funcionalidades

- Formularios de recepción, producción, envasado, control pesado y expendio
- Gestión de registros con filtros
- Gestión de usuarios
- Panel de administración
